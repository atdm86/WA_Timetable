Git commit and push workflow
1. make changes to the project
2. Open git kraken
3. Stage files with chenges
4. add a commit message
5. commit
6. push!!! very important
7. if unsure check on github that changes have appeared

Git Branching workflow
1. Click 'branch'
2. Name your branch - make the name descriptive of what you intend to do.
3. Make changes to project. This is what we are doing now
4. Commit changes, see workflow above.
5. When finished work on branch merge it into 'master'
